# Laba2

## Установка и запуск

### 1. Создание файла окружения
Создайте файл `.env` на основе шаблона:

```bash
cp .env.example .env
```

После этого отредактируйте файл `.env`, добавив необходимые значения .

### 2. Сборка и запуск проекта
Соберите и запустите контейнеры с помощью Docker Compose:

```bash
docker compose up
```

Эта команда запустит все сервисы в фоновом режиме.

### 3. Восстановление применение миграций
Примените миграции

```bash
docker exec laba2-web-1 alembic upgrade head
```

Убедитесь, что контейнер `laba2-db-1` запущен перед выполнением команды
